Sovereign of the Blooming Suns

EMPEROR OF THE FADING SUNS INSPIRED LÖVE GAME

This is an attempt to develop a 4X Strategy Game featuring detailed planet-side mechanics, a la Emperor of the Fading Suns using LÖVE, a framework for making 2D games in the Lua programming language.

  • Lead: Edward Miller
  • https://github.com/coderbunker/blooming-suns

The Human Computer

ONLINE METEORJS PROJECT TO RUN WORKSHOP

This is a MeteorJS project to run workshops for large class settings (30 to 100 students) to demonstrate how to computers work in participatory exercises using party hats to represent binary state (1 or 0). Complement to the "Build a computer" Coursera course.

  • Lead: Ricky Ng-Adam
  • https://www.youtube.com/watch?v=JK6ZPzD_3P0
  • https://github.com/coderbunker/human-computer
