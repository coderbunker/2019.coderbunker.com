Blockchain

Engineering Blockchain Networks

The idea of the emulated blockchain network came from the idea that some technologies exist to publish smart contracts on an existing blockchain. But to develop and deploy a new blockchain from scratch nothing really exists. This ambitious project is just starting, it aims at using existing tools such as docker mongoDb, golang to create a really customised blockchain.

Opensource Software Development on Github

Do you sometimes find yourself at the terminal entering magic git commands while wondering how git actually works? Do you struggle to collaborate with other developers working on the same github project? In this workshop, we’ll be building an open source web application on github. You’ll learn some useful git commands, how to introduce your code into the application while working in parallel with other developers, how to review each other's code for bugs, and how automate testing your code for breaking changes.
